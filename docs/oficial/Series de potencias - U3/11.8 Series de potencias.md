# 11.10 Series de Potencias

Una serie de potencias es una serie de la forma
$$1 \qquad \sum_{n=0}^{\infty} c_n x^n = c_0 + c_1 x + c_2 x^2 + c_3 x^3 + \cdots$$donde ($x$) es una variable y las ($c_n$) son constantes llamados coeficientes de la serie. Para cada ($x$) fija, la serie ($1$) es una serie de constantes que podemos probar para ver si son convergentes o divergentes. Una serie de potencias podrÃ­a ser convergente para algunos valores de ($x$) y ser divergente para otros. La suma de la serie es una funciÃ³n
$$f(x) = c_0 + c_1 x + c_2 x^2 + \cdots + c_n x^n + \cdots$$
cuyo dominio es el conjunto de todas las ($x$) para las cuales la serie converge. Observe que ($f$) es anÃ¡loga a una funciÃ³n polinomial. La Ãºnica diferencia es que ($f$) tiene un infinito de tÃ©rminos.
****
Por ejemplo, si tomamos ($c_n = 1$) para toda ($n$), la serie de potencias se transforma en una serie geomÃ©trica$$\sum_{n=0}^{\infty} x^n = 1 + x + x^2 + \cdots + x^n + \cdots$$que es convergente cuando $-1<x<1$ y es divergente cuando $|x| \ge 1$. (ver mis notas p.3)

#### **Teorema. Para una serie de potencias dada** $$\sum_{n=0}^{\infty} c_n(x-a)^n$$hay sÃ³lo tres posibilidades:
1) La serie converge sÃ³lo cuando ($x=a$).
2) La serie converge para toda ($x$).

El **intervalo de convergencia** de una serie de potencias es el intervalo que consiste en todos los valores de ($x$) para los cuales la serie converge. 
- En el caso 1) el intervalo consta de un solo punto ($a$).

##### **ObservaciÃ³n**. notar que si $$\sum_{n=0}^{\infty} c_n(x-a)^n$$converge en algÃºn ($x_0 \neq a$), entonces por (3) del teorema anterior ($R \ge |x_0 - a|$), y ademÃ¡s la serie converge $\forall x$ tal que ($|x-a| < |x_0-a|$).

**En otras palabras.** Si alguien nos dice que existe un $x_0 \neq a$ para el cual la serie converge, entonces el radio de convergencia $R$ debe ser mayor o igual que la distancia entre $a$ (el centro de la serie) y $x_0$. 
AdemÃ¡s intuimos que la serie converge para todos los $x$ que estÃ©n a una distancia menor a la distancia del $x_0$ al $a$.
Luego si la serie diverge en $x_1$, por lo dicho anteriormente, el radio $R$ debe ser menor o igual que la distancia entre $x_1$ y $a$.

#### **Teorema (Crit. del cociente para series de potencias)**: Dada la serie de potencias
$$\sum_{n=0}^{\infty} c_n(x-a)^n$$
con ($c_n \neq 0 \quad \forall n \ge n_0$) y ($R$) su radio de convergencia. Escribimos
$$L \doteq \lim_{n \to \infty} \frac{|c_{n+1}|}{|c_n|}$$
(1) Si ($0 < L < \infty$), entonces ($R = \frac{1}{L}$)
(2) Si ($L = 0$), entonces ($R = \infty$)
(3) Si ($L = \infty$), entonces ($R = 0$).
